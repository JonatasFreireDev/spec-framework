# Prompts

## Purpose

Store reusable prompts for skills, reviews, audits, and artifact generation.

## When To Use

Use this folder when a prompt is stable enough to reuse and aligns with the architecture in `FRAMEWORK.md`.

## Expected Files

- `README.md`: folder purpose and usage.
- Future prompt files for architecture review, feature generation, readiness audits, release audits, and documentation updates.
