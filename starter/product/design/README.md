# Design

## Purpose

Store product design references, mockups, screenshots, and design-system notes.

Canonical use-case contracts remain in each use case's `design.md`. Versioned external sources live in `sources/DSRC-NNN/`; derived wireframes, mockups, prototypes, mappings, and evidence live in `use-cases/<slug>/`. Design prototypes are non-production artifacts.

The shared product Design System lives in `system/`. Keep it draft until foundations, tokens, components, patterns, accessibility, compatibility, and sources are reviewed and explicitly approved.

## Next Step

Create design artifacts after a Specification exists and the use case has a UI surface.
