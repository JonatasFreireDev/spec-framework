# Product Knowledge

## Purpose

Store product-specific knowledge. This folder is for the adopting product, not for framework method history.

## Expected Areas

| Area | Purpose |
| --- | --- |
| `business-rules/` | Product business rules. |
| `conventions/` | Product-specific conventions such as technical gates and security baseline. |
| `decisions/` | Product decisions only. |
| `examples/` | Product examples. |
| `glossary/` | Product vocabulary. |
| `patterns/` | Product patterns. |
| `prompts/` | Product prompts. |
| `templates/` | Installed framework templates or product overrides. |

## Boundary Rule

Framework method decisions belong to the framework repository as FDRs, not here.
