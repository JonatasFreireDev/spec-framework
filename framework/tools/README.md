# Framework Tools

## Purpose

Framework utilities are commands in the compiled `spec-framework` CLI.

## Installed Tools

| Tool | Purpose |
| --- | --- |
| `spec-framework move` | Move an artifact safely while rewriting resolvable Markdown links and JSON paths. |

The release binary also provides `init`, `validate`, `upgrade`, and `version`.

Framework development tests remain in `framework/tests/` and are not copied into adopter repositories.
