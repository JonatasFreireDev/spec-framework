# QR Code Check-in Feature

## Purpose

QR Code Check-in lets an event attendee prove event participation and lets an organizer validate that proof.

## When To Use

Use this feature when work concerns QR token generation, display, validation, expiration, idempotent attendance marking, organizer scanning, or check-in analytics.
