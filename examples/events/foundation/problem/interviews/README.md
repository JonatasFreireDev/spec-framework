# Interviews

## Purpose

Store interview notes used to validate the problem, user segment, current workaround, and emotional or operational cost of the pain.

## When To Use

Use this folder for direct user, stakeholder, customer, or operator conversations.

## Expected Files

- `YYYY-MM-DD-participant-or-segment.md`: one interview note per conversation.
- Participant segment, context, key quotes, observed pain, workaround, severity, follow-up questions, and consent notes when applicable.

Primary owner: Problem Discovery AI.
