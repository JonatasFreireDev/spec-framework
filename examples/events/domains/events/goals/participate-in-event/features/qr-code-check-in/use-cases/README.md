# QR Code Check-in Use Cases

## Purpose

This folder stores the executable interactions for QR Code Check-in. Each use case gets its own folder with context, behavior, specification, design, plan, graph, tasks, tests, analytics, and audit.

## When To Use

Use this folder when adding or auditing an interaction that can be validated end to end.

## Expected Files

- `attendee-checks-in-with-qr-code/`: attendee presents or refreshes QR proof.
- `organizer-validates-qr-code/`: organizer scans and validates QR proof.

## Responsible Skill

Primary owner: Use Case AI.

## Next Step

Keep each use-case folder self-contained and traceable to the feature context.
